Welcome to Pokémon Unity

Pokémon Unity is a project that was developed by IIcolour Spectrum, the project is now open source. As the project is not 100% completed, hopefully more and more people will contribute to develop a stable build for Pokémon Unity. Many lines of code still needs to be written and many assets still needs to be designed.

Template

Pokémon Unity isn't just an open source project, feel free to clone this project and use it as a template to start developing your own Pokémon game!
